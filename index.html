Core Engine Loop

function engineTick():
    // 1. Gather signals
    rtScore      ← RTAnalyser.getScore()           // 0..1
    moveScore    ← MovementAnalyser.getScore()     // 0..1
    stressIndex  ← StressDetector.getStressIndex() // 0..1

    // 2. Weighted fusion
    raw ← 0.40 × rtScore + 0.35 × moveScore − 0.25 × stressIndex
    raw ← clamp(raw, 0.1, 1.0)

    // 3. EMA smoothing
    difficulty ← difficulty × (1 − 0.15) + raw × 0.15

    // 4. Hysteresis — only change zone if delta > 0.08
    if |difficulty − lastZoneDifficulty| > 0.08:
        zone ← mapToZone(difficulty)
        lastZoneDifficulty ← difficulty

    // 5. Rescue mode check
    if stressIndex > 0.85 for > 90s:
        activateRescueMode()

    // 6. Map to enemy parameters
    enemyParams ← EnemyAI.getParams(difficulty, zone)
    return { difficulty, zone, enemyParams }

Transparency

Players can see their own signal scores and how difficulty is being adjusted. No hidden manipulation — the dashboard is the proof.

🎛
Player Agency

Manual difficulty override always available. NDS adapts, but the player has final control — including the ability to disable adaptation entirely.

🔒
Data Minimization

Only behavioural signals are processed — no biometric data leaves the device. All processing is local, real-time, and ephemeral.

⚖
Fairness

The NDS system ensures difficulty adapts to individual skill, not demographic assumptions. Every player gets a personalized challenge curve.
